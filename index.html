Hex Grid

Summary

Being inspiried by games like Heroscape, Civilization, and Black & White, I wanted to create a dynamic hex grid system that would allow for the user to build out and modify a world comprimising of hex tiles. The user would be abkle to create pathways and modify tile height.

Architecture

The main components that need to be in place for this to succeed is a grid of tiles that each individually store environment related data as well as know their neighbors and can change it's own data based off changes within those neighbors.
